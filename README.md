## Installation

- Clone this repository on your local computer.
- Build docker image.
- Run docker image.

```shell
git clone https://github.com/jimmy12613/LAMP_PROJECT.git
cd LAMP_PROJECT/
docker-compose up
```

LAMP_PROJECT is now ready,\
Network interface via `http://127.0.0.1:8001`\
phpMyAdmin via `http://127.0.0.1:8000`
